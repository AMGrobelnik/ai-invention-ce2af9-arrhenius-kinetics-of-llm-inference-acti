Schema: question_id (int64), question (str), options (List[str], 10 items A-J), answer (str A-J), answer_index (int64 0-9), cot_content (str), category (str), src (str). Baseline random accuracy 10% (not 25%). Official prompt ends with 'Think step by step and then output the answer in the format of The answer is (X)'. For single-token logprob extraction: use max_tokens=1, top_logprobs=20, and take max(logprob('X'), logprob(' X')) for each letter to handle the space-prefix tokenization ambiguity. Handle variable option counts by only considering letters up to len(options). TURN (Du et al. 2025) computes dataset-level temperature via entropy inflection: t_j=j*0.1 grid, H computed over top-1000 tokens, l(t_j)=log H(t_j), inflection j* = first j where Δ²l(j)>0, T_pred = j*×0.1 + beta (0 majority, 0.1 best-of-N); dataset-level only, tested on MATH/MBPP generation tasks. For MCQA adaptation: use first-token entropy over A-J letters from top_logprobs. Wu et al.
