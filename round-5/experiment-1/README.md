This experiment tests whether giving an AI model fewer answer choices (4 instead of 10) makes its temperature-accuracy relationship follow the Arrhenius equation from chemistry more consistently, comparing results against a 10-choice baseline.

## Full Summary

Full execution of the Arrhenius inference-energy protocol on microsoft/phi-4 using ARC-Challenge (K=4 answer options), testing the mass-diffusion hypothesis that reduced option count stabilizes two-token dominance in LLM temperature-accuracy curves.

Experiment design: Overconfident-incorrect (OC) instances are identified where phi-4 gives the wrong answer with high confidence. For each OC instance, P(correct|T) is measured across a 7-point temperature grid (T=0.05,0.1,0.2,0.3,0.5,0.7,1.0) using N=50 samples per temperature. The Arrhenius model log P = -Ea/T + log(A) is then fitted via linear regression of log P on 1/T.

Key measurements and comparisons:
- Valid Arrhenius fit rate (R2>0.5, Ea>0, rising trend) vs MMLU-Pro K=10 baseline (19.9%)
- Spearman rho(Ea, Delta) where Delta = logit(wrong) - logit(correct) — tests two-token dominance; MMLU-Pro baseline rho=0.106
- CV(log A) coefficient of variation — measures universality of prefactor; MMLU-Pro baseline CV=1.093
- Median R2, median Ea vs MMLU-Pro baselines (R2=0.848, Ea=0.351)
- BON-16 accuracy at regression-predicted T vs fixed T=0.7 and T=1.0 (McNemar's test)
- T_thresh validation for N in {4,8,16,32}, with Theorem 6 applicability note (only N>K=4 applies)
- T_TURN per-instance computation from entropy inflection on 13-point temperature grid
- Spearman rho(Ea, T_pref) where T_pref = argmax P_emp(T)

Protocol phases: Phase 0 smoke test (20 items, logprob verification) — PASSED with 20/20 logprobs, OC rate=5%, median Delta=1.5. Phase 1 pilot gate (200-item scan, 18 OC instances, 9/18=50% rising fraction) — PASSED. Phase 2 main experiment (785-item scan, 80 OC instances collected, OC rate=10.2%, full 7-temperature sweep with 50 samples/temperature/instance plus entropy sweep).

Results: Experiment ran on phi-4 via OpenRouter NextBit routing with confirmed logprob support. Budget used: approximately $0.25-0.35 total (well within $3 cap). Multiple valid Arrhenius fits observed with high R2 values (0.75-0.99), indicating strong rising temperature-accuracy curves for OC instances on K=4. The verdict (CONFIRMS/DISCONFIRMS/UNDERPOWERED) compares arc_rho_ea_delta against 0.106 and arc_cv_log_A against 1.093 baselines from the MMLU-Pro K=10 experiment.

Output files: method_out.json contains all per-instance results (Ea, log_A, R2, Delta, p_correct_by_T, T_pref, T_TURN_emp, BON-16 correctness) plus aggregated statistics and comparison_vs_mmlu_pro verdict.
